Etiquetas semántica estructurales

DIV: Es una etiqueta contenedora que no tiene semántica
HEADER:Cabecera de un sitio web o una sección
MAIN:define la sección principal del documento, solo puede existir una por documento
FOOTER: pié de pagina de un sitio web o una sección
NAV: representa un menu de navegación
ARTICLE: representa una sección de autocontenido (que por si sola se aplica)
ASIDE: representa contenido complementario o secundario
SECTION: representa una seccion de contenido generico
ADDRESS: representa una direccion física
